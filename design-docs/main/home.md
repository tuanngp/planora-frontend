## Style Guidelines

### Typography
- Headlines: Montserrat Bold
  - Size: 48px/56px
  - Color: #111827
- Body: Inter Regular
  - Size: 16px/24px
  - Color: #4B5563

### Color Palette
- Primary Gradient: #4F46E5 → #818CF8
- Secondary: #10B981
- Accent: #F59E0B
- Text: #111827, #4B5563
- Backgrounds: #FFFFFF, #F9FAFB

### Components
- Buttons:
  - Primary: Solid gradient
  - Secondary: Outlined
  - Ghost: Transparent
- Cards:
  - Shadow: 0 4px 6px -1px rgba(0,0,0,0.1)
  - Radius: 12px
  - Hover lift: transform: translateY(-4px)

- High-quality travel photos

- Optimized loading
